# 在线编辑器工具集需求文档

## 1. 应用概述

### 1.1 应用名称
在线编辑器工具集

### 1.2 应用描述
一款集成 Markdown、PlantUML 和 Mermaid 在线编辑功能的 web 工具平台，支持实时预览效果，帮助用户快速完成文档编写和图表绘制。\n
## 2. 核心功能

### 2.1 Markdown 在线编辑器
- 提供左右分屏编辑预览模式
- 左侧为 Markdown 编辑区域，支持语法高亮和代码提示
- 右侧为实时渲染预览区域，展示 Markdown 效果
- 支持常用 Markdown 语法：标题、列表、链接、图片、代码块、表格等
- 提供导出功能，支持导出为 HTML 或 PDF 格式

### 2.2 PlantUML 在线编辑器
- 提供独立的编辑区域，支持 PlantUML 语法输入
- 点击生成按钮后实时渲染生成对应的 UML 图表
- 支持多种 UML 图类型：类图、时序图、流程图等
- 提供图表导出功能，支持导出为 PNG 或 SVG 格式

### 2.3 Mermaid 在线编辑器
- 提供独立的编辑区域，支持 Mermaid 语法输入
- 点击生成按钮后实时渲染生成对应的图表
- 支持多种图表类型：流程图、时序图、甘特图、思维导图等
- 提供图表导出功能，支持导出为 PNG 或 SVG 格式

## 3. 页面结构

### 3.1 首页
- 导航栏：包含 Markdown 编辑器、PlantUML 编辑器、Mermaid 编辑器三个入口
- 每个编辑器入口展示功能介绍和快速开始指引

### 3.2 Markdown 编辑器页面
- 左侧编辑区域：输入 Markdown 文本
- 右侧预览区域：实时展示渲染效果
- 顶部工具栏：包含格式切换、导出功能等

### 3.3 PlantUML 编辑器页面
- 上方编辑区域：输入 PlantUML 语法
- 下方预览区域：展示生成的 UML 图表
- 顶部工具栏：包含生成按钮、导出功能等

### 3.4 Mermaid 编辑器页面
- 上方编辑区域：输入 Mermaid 语法
- 下方预览区域：展示生成的图表
- 顶部工具栏：包含生成按钮、导出功能等
